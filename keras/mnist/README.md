# MNIST Model


### Steps:

* Install Python3 Libraries requirements:
```bash
$ ./install_requirements.sh
```

* Get data from MNIST url run:
```bash
$ ./download_data.sh
```

* Test loading data from files run:
```bash
$ ./load_data.py
```

* Create, Compile, Train, Validate Model
```bash
$ ./create_model.py
```

* View model
```bash
$ ./view_model.py 
```
